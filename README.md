# SmartClinic

SmartClinic is a PHP-based clinic management system developed to manage appointments, reports, and clinic-related operations. 
for database import check sql file inside of sql folder.

---

## 🚀 Features

- Patient Appointment Management
- Doctor Appointment Scheduling
- PDF Report Generation using FPDF
- Dashboard Management
- Form Validation
- Database Integration using MySQL
- Responsive UI

---

## 🛠️ Technologies Used

- PHP
- MySQL
- JavaScript
- jQuery
- HTML5
- CSS3
- Bootstrap
- FPDF
